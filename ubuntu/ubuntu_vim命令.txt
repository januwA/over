i进入编辑模式 
esc推出编辑模式
 ：q！不保存退出 
:wq保存退出
命令模式：两次敲击d删除一行代码
	o 在光标所在的行下方插入并且进入输入模式
	yy 复制所在的行
	p在下方粘贴（小写
	P在上方粘贴（大写
	vim +  文件名 ：表示打开该文件把光标定位在最后一行
	vim +3  文件名 ：表示打开该文件把光标定位在3行
	vim +/word  文件名 ：表示打开文件把光标定位在word出现第一次的那一行 n来回切换

	vim [文件名] [文件名] [文件名]： 表示一次打开或创建多个文件
	地行模式  :n  切换到下一个文件
	地行模式  :N  切换到上一个文件
	地行模式  :ls  查看当前打开的所有文件
	地行模式  :15  光标定位到15行
	地行模式  /word  从光标开始向后搜索 word广健词
	地行模式  ？word  从光标开始向前搜索 word广健词
	set nu   // 打开行号
	sp [文件名] 可以打开多个文件  ctrl+w 上下箭头可跳转
